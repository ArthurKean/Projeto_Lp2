# 📚 Sistema de Gerenciamento de Extensão - UFMA

## 📌 Descrição

Este projeto consiste na implementação de um sistema de gerenciamento de atividades de extensão para alunos do curso de Ciência da Computação da UFMA.

O sistema foi desenvolvido como parte da disciplina **Linguagem de Programação II (LP2)**, ministrada pelo professor Geraldo.

---

## 🎯 Objetivo

Desenvolver um sistema em Java aplicando conceitos de Programação Orientada a Objetos (POO), com foco em organização de código e separação de responsabilidades.

---

## 📂 Estrutura do Projeto

* `model` → Entidades do sistema

    * `enums` → Tipos enumerados
* `service` → Regras de negócio

---

## ⚙️ Etapas do Projeto

* Análise da modelagem fornecida
* Estruturação em pacotes
* Implementação das entidades e serviços
* Testes básicos das funcionalidades

---

## 🚀 Funcionalidades

* Gerenciamento de oportunidades de extensão
* Associação de autores às oportunidades
* Uso de enums para padronização de estados e tipos
